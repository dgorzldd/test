[ 발사믹 사용법 ]
1. 설치후 라이센스키를 입력한다
2. 화면에 구성요소를 드래그한 후 단축기
(1) ctrl + 스크롤 : 화면 축소/확대
(2) ctrl + 0 : 화면 맞춤
(3) ctrl + 1 : 원래 크기로
(4) 스페이스바 드래그 : 화면끌기
(5) alt + 드래그&드롭 : 복사하기
(6) ctrl + Enter : 글자 편집 (시작/끝)


3. 조작반응들
(1) 스테이지 요소를 클릭시 색 + 조저럼 생김
-> 선택안되게 하기 : 오른쪽 버튼 메뉴
    Lock... 잠그기 <-> Unlock... 풀기
(2) 스테이지 겹쳐진 요소를 드래그 이동시
-> 중심점 보조선이 해당위치에서 나타남
-> 가운데 놓고 싶을때 선 나올때 놓음

4. 와이어프레임 2가지 모드
(1) 스케치 모드 : 손그림처럼 출력
(2) 와이어프레임 모드 : 깔끔하게 출력
-> 선택은 오른쪽 상단 
->> 프로젝트 정보보기버튼
->>> 하단 선택버튼(Sketch / Wireframe)
